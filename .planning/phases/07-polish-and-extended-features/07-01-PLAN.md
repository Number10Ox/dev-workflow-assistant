<task type="auto">
  <name>Task 2: Create diagnostic code system and content hashing</name>
  <files>src/google-docs/diagnostics.js, src/google-docs/hash-content.js, test/google-docs/diagnostics.test.js, test/google-docs/hash-content.test.js</files>
  <action>
1. Create `src/google-docs/diagnostics.js`:
   - `ImportDiagnostic` class with constructor(code, level, message, elementId)
     - code: string (e.g., 'DWA-GDOC-201')
     - level: 'info' | 'warning' | 'error' | 'fatal'
     - message: string
     - elementId: optional string (Google Docs element ID)
     - toString() method: `{code} [{LEVEL}] {prefix}{message}`
   - `DIAGNOSTICS` object with predefined codes:
     - Info (100-series): TOC_DROPPED (100), PAGE_BREAK_CONVERTED (101), HEADING_NORMALIZED (102)
     - Warning (200-series): IMAGE_PLACEHOLDER (201), TABLE_TO_HTML (202), COMMENT_DROPPED (203), EQUATION_PLACEHOLDER (204), FOOTNOTE_SIMPLIFIED (205), LINK_BROKEN (206)
     - Error (300-series): INVALID_TABLE_STRUCTURE (301), ELEMENT_PARSE_FAILED (302)
     - Fatal (400-series): AUTH_FAILURE (401), DOC_NOT_FOUND (402), CORRUPT_DOCUMENT (403), MCP_UNAVAILABLE (404)
   - `createDiagnostic(type, vars = {})` - Factory function
     - type: key from DIAGNOSTICS object
     - vars: object with template variables (e.g., { imageId: 'img_001' })
     - Returns ImportDiagnostic instance

2. Create `src/google-docs/hash-content.js`:
   - `hashImportedContent(markdownContent)` - Returns `sha256:{64-char-hex}`
     - Uses Node's crypto.createHash('sha256')
     - UTF-8 encoding
   - `verifyContentHash(currentContent, storedHash)` - Returns { unchanged: boolean, currentHash, storedHash }
   - `extractImportRegion(markdownContent)` - Returns content between DWA:IMPORT_BEGIN and DWA:IMPORT_END markers
     - Returns null if markers not found
     - Uses regex matching for marker detection

3. Create tests:
   - `test/google-docs/diagnostics.test.js`:
     - Test ImportDiagnostic toString() format
     - Test createDiagnostic with template variables
     - Test all severity levels
   - `test/google-docs/hash-content.test.js`:
     - Test hashImportedContent produces consistent hashes
     - Test verifyContentHash detects changes
     - Test extractImportRegion with valid/missing markers
  </action>
  <verify>
`npm test -- --test-name-pattern="diagnostics|hash-content"` passes all tests.
All exports available: ImportDiagnostic, DIAGNOSTICS, createDiagnostic, hashImportedContent, verifyContentHash, extractImportRegion.
  </verify>
